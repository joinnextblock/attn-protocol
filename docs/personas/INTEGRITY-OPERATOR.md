## ROLE

As an Integrity Operator in the PROMO Protocol ecosystem, your primary function is detecting and preventing attention fraud. You maintain advanced fraud detection systems that identify suspicious patterns, fake engagement, and manipulation attempts in the attention marketplace. You can set fraud detection standards and implement verification requirements for brokerages. The specific methods you use for fraud detection are entirely at your discretion - you can implement any verification technique you deem effective, from basic pattern recognition to sophisticated AI-powered fraud detection systems. When fraud is detected, you publish Fraud Detection events that help maintain the protocol's integrity. This role allows you to innovate while protecting the ecosystem from bad actors and ensuring genuine attention engagement.

## RESPONSIBILITIES

- Implement and maintain fraud detection systems
- Identify suspicious attention patterns
- Detect automated engagement attempts
- Monitor for coordinated fraud attacks
- Validate attention tracking authenticity

## REQUIREMENTS

- Advanced fraud detection infrastructure
- AI/ML-powered pattern recognition systems
- Real-time monitoring and analytics
- Secure event publishing system
- Advanced security measures
- Backup and redundancy systems

## INCENTIVES

- Revenue share from fraud prevention fees
- Reputation building through reliable detection
- Network effects from increased protocol trust
- Potential for premium fraud detection services
