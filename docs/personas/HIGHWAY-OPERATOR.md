## ROLE

As a Highway Operator in the PROMO Protocol ecosystem, your core function is providing the best experience for discovering billboards across relays. You maintain a comprehensive view of all available promotional spaces by tracking relay announcements and aggregating billboard data. You implement powerful discovery tools that help users find billboards based on their preferences, from location and type to availability and historical performance. The specific discovery methods you implement are at your discretion - from basic search to advanced recommendation systems. This focused role allows you to provide a reliable service for discovering billboards in the decentralized attention marketplace.

## RESPONSIBILITIES

- Monitor and display billboards across designated relays
- Track and aggregate relay announcements
- Implement intuitive billboard discovery tools
- Provide personalized billboard recommendations
- Maintain system uptime and reliability
- Report technical issues and system performance

## REQUIREMENTS

- Technical infrastructure for relay monitoring
- Reliable data aggregation mechanisms
- Secure event processing system
- Monitoring and analytics capabilities
- Backup and redundancy systems

## INCENTIVES

- Subscription fees from users accessing billboard data
- Premium features for advanced filtering and analytics
- API access fees for automated billboard discovery
- Reputation building through reliable service
- Network effects from increased protocol usage
