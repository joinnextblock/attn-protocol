## ROLE

As an API Operator in the PROMO Protocol ecosystem, your core function is providing standardized API endpoints for protocol interactions. You maintain infrastructure that enables seamless communication between different protocol participants, ensuring high availability and performance. You implement a consistent set of functions that developers can use to build PROMO Protocol applications, made available through the DVMCP (Decentralized Virtual Machine Context Protocol). The specific implementation details are at your discretion - you can use any technology stack and architecture that meets the protocol's requirements. This flexibility allows you to optimize for your specific use case while maintaining protocol compatibility.

## RESPONSIBILITIES

- Implement and maintain standardized API endpoints
- Ensure high availability and performance
- Monitor system health and performance
- Implement security measures and rate limiting
- Handle API versioning and updates
- Report technical issues and system metrics
- Provide consistent function interfaces
- Document API specifications
- Maintain DVMCP compatibility

## REQUIREMENTS

- Robust API infrastructure
- Monitoring and logging systems
- Security implementation
- Load balancing capabilities
- Backup and redundancy systems
- Standardized function implementations
- API documentation
- DVMCP protocol support

## INCENTIVES

- Revenue from API usage
- Network participation benefits
- Reputation through reliable service
- Potential for premium API features
